Playlist Shortcuts

Ctrl+ASelect AllSelect the entire playlist
Ctrl+CCopyCopy the selection and put it on the Clipboard
Ctrl+NPlaylist NewCreate a new playlist
Ctrl+OPlaylist OpenOpen an existing playlist
Ctrl+SPlaylist SaveSave the current playlist
Ctrl+VPasteInsert Clipboard contents
DeleteDeleteDelete the current selection
InsertInsert PatchInsert one or more patches into the playlist
Ctrl+XCutCut the selection and put it on the Clipboard
